# Traffic Demand Forecasting using Ensemble Learning

## Overview
Developed a traffic demand prediction system using real-world transportation booking data from Flipkart Gridlock Hackathon 2.0.

## Technologies Used
- Python
- LightGBM
- CatBoost
- Scikit-Learn
- Pandas
- NumPy

## Features
- Historical lag feature engineering
- Geo-temporal statistical features
- Target encoding for categorical variables
- 5-fold cross-validation
- Ensemble learning

## Results
- Secured Rank 202 on the Flipkart Gridlock Hackathon leaderboard.
- Evaluated using R², MAE, MSE, and RMSE metrics.

## Repository Structure
- `Flipkart.ipynb` : Complete model development and evaluation notebook.
